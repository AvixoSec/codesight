How it works

Three commands. No config files.
No build steps. No YAML.

Install, point at a file, read the output.

01

Install

pip install codesight and set your API key. 20 seconds. Or use Ollama for zero-config offline mode.

02

Pick a task

review, bugs, security, docs, explain, or refactor. Each one has a tuned system prompt.

03

Fix what matters

Structured findings with line numbers, severity, CWE IDs. SARIF output for CI. Exit codes for pipelines.

What it finds

Linters check syntax.
This checks logic.

Your linter won't tell you that the auth flow leaks tokens or that there's a race condition in the queue handler. CodeSight understands what the code does and where it breaks.

SQL injection Hardcoded secrets Race conditions Off-by-one Resource leaks Dead code paths Auth bypass SSRF Deserialization
