How It Runs

1. Scanner finds breadth

Semgrep or CodeQL writes SARIF with file paths, line numbers, rules, and messages.

2. CodeSight adds context

Paths are resolved under the source root and local snippets are attached safely.

3. Judge checks evidence

Optional judge and skeptic passes ask for source, sink, missing guard, path, and fix.

4. CI gets a verdict

Findings become exploitable, likely exploitable, uncertain, or dismissed.
