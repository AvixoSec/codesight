Benchmarks

The current benchmark is a smoke test. Treat it that way.

Current Dataset

It has 10 vulnerable Python cases and 2 clean traps. It checks provider behavior, CWE extraction, and basic false-positive pressure.

CaseExpected CWE

Run

Terminal
$ codesight benchmark --models gpt-5.4 llama3
$ codesight benchmark --json > benchmark-results.json

Measured

  • detection rate
  • true positives and false negatives
  • false positives on clean cases
  • response time
  • token usage
  • provider errors

Not Measured Yet

  • exploitability verdict quality
  • source-to-sink evidence quality
  • SARIF alert verification
  • framework-specific behavior
  • multi-file context
  • repeatability across multiple runs

Claim Rule

Do not claim CodeSight is better than another tool unless the repo includes the cases, command, expected verdicts, raw results, model, provider, and run date.
