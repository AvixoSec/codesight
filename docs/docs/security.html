Security Policy

CodeSight is a defensive tool. Report issues privately.

Report A Vulnerability

Email security@example.com. Do not open a public issue for a vulnerability in CodeSight itself.

Include the affected version, steps to reproduce, impact, and any suggested fix.

Data Handling

  • Ollama keeps source local.
  • Hosted providers receive the selected source context.
  • Preview mode shows context before judge mode sends it.
  • Project config cannot set API keys, base URLs, or default provider.
  • User keys live in user config or keyring, not repo config.

Scope

CodeSight reviews code and scanner alerts. It does not execute untrusted project code, exploit targets, or exfiltrate repository data.
