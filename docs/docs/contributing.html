Contributing

CodeSight is open source. Here's how to contribute.

Setup

Terminal
$ git clone https://github.com/AvixoSec/codesight.git
$ cd codesight
$ python -m venv .venv
$ source .venv/bin/activate
$ pip install -e ".[dev]"

Tests

Terminal
$ pytest tests/ -v

Linting

Terminal
$ ruff check codesight/
$ ruff format codesight/
$ mypy codesight/ --ignore-missing-imports

Pull Request Process

  1. Fork the repository
  2. Create a feature branch: git checkout -b feat/my-feature
  3. Make your changes with clear commit messages
  4. Ensure tests pass and linting is clean
  5. Open a PR against main

What We're Looking For

  • New provider integrations
  • Improved system prompts for better detection rates
  • Language-specific compression patterns
  • Bug fixes with test coverage
  • Documentation improvements

Reporting Issues

Use GitHub Issues. Include your Python version, OS, reproduction steps, and the full error traceback.
