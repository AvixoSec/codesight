Commands

Use the UI for day-to-day work. Use commands when you want scripts and CI.

Guided UI

Terminal
$ codesight
$ codesight ui

The UI can scan files, scan folders, review diffs, verify SARIF, write artifact bundles, and run judge mode.

Single File

Terminal
$ codesight security src/auth.py
$ codesight review src/auth.py
$ codesight bugs src/auth.py
$ codesight explain src/auth.py
$ codesight refactor src/auth.py

Folder Scan

Terminal
$ codesight scan src --task security
$ codesight scan src --task review --ext .py .js
$ codesight scan src --task security --estimate
$ codesight scan src --task security --output sarif > codesight.sarif

Git Diff

Terminal
$ codesight diff --task security
$ codesight diff --task review --output markdown

Verify SARIF

Terminal
$ codesight verify semgrep.sarif --source . --output markdown
$ codesight verify semgrep.sarif --source . --preview-context
$ codesight verify semgrep.sarif --source . --judge --skeptic --profile auto
$ codesight verify semgrep.sarif --source . --artifact-dir .codesight-proof

Benchmarks

Terminal
$ codesight benchmark
$ codesight benchmark --models gpt-5.4 llama3
$ codesight benchmark --json > benchmark-results.json

Config And Health

Terminal
$ codesight config
$ codesight health
$ codesight health --provider openai

Global Options

OptionMeaning
-p, --providerProvider label from config or a built-in provider.
-o, --outputmarkdown, json, plain, or sarif.
--langCLI message language.
-v, --versionShow version.

Exit Codes

  • 0 - completed without blocking findings
  • 1 - warnings or threshold hit
  • 2 - critical security finding or blocking verify verdict
