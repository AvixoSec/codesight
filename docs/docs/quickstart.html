Quick Start

Install, open the guided UI, then run a scan or verify scanner SARIF.

1. Install

Terminal
$ pip install codesight

2. Open The UI

This is the easiest path when you do not know the flags yet.

Terminal
$ codesight

3. Scan A File

Terminal
$ codesight security src/app.py
$ codesight review src/app.py
$ codesight bugs src/app.py

4. Scan A Folder

Terminal
$ codesight scan src --task security
$ codesight scan src --task security --output sarif > codesight.sarif

5. Verify SARIF

Terminal
$ semgrep scan --config auto --sarif > semgrep.sarif
$ codesight verify semgrep.sarif --source . --output markdown

6. Use Local Mode

Terminal
$ ollama serve
$ codesight security src/app.py --provider ollama
