# Code of Conduct

## The Short Version

Be respectful. Be constructive. Don't be a jerk.

## The Longer Version

This project is maintained by a small team. We want participation to be a positive experience for everyone.

**Expected behavior:**
- Constructive feedback with specific suggestions
- Respect for different experience levels and backgrounds
- Focus on what's best for the project and its users

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Publishing private information without consent
- Sustained disruption of discussions

## Enforcement

Report violations through the project's GitHub issue tracker or maintainer profile. We'll review and take appropriate action, up to and including banning from the project.
